# 🌿 Cruz Maya — Calculadora de Nacimiento Tzolkin

Herramienta web estática para calcular la **Cruz Maya** de cualquier persona según su fecha de nacimiento, basada en el calendario sagrado **Tzolkin**.

## ✨ Demo

Abre `index.html` en cualquier navegador moderno — **no requiere internet ni servidor**.

## 📐 ¿Qué calcula?

A partir de una fecha de nacimiento, la herramienta determina los **9 puntos de la Cruz Maya**:

| Posición | Offset | Significado |
|----------|--------|-------------|
| Nacimiento | — | Esencia y personalidad |
| Origen | −9 | Raíces, de dónde vienes |
| Destino | +9 | Misión de vida |
| Izquierda | −7 | Lo femenino, desafíos |
| Derecha | +7 | Lo masculino, dones |
| 4 Auxiliares | ±1/2 | Energías de soporte |

Cada posición muestra el **número** (1–13) y el **Nawal** (uno de los 20 días sagrados) con su nombre en K'iche', Yucateco y Nahuatl.

## 🗓️ Los 20 Nawales del Tzolkin

`Imox · Iq' · Aq'ab'al · K'at · Kan · Keme · Keej · Q'anil · Toj · T'zi' · B'atz' · E · Aj · I'x · Tz'ikin · Ajmaq · No'j · Tijax · Kawoq · Ajpu`

## 🚀 Uso

1. Clona o descarga el repositorio
2. Abre `index.html` en tu navegador
3. Ingresa nombre y fecha de nacimiento
4. Presiona **Calcular Cruz**

### GitHub Pages

Puedes activar **GitHub Pages** en la rama `main` para tener una URL pública:
`Settings → Pages → Branch: main → / (root)`

## 🖨️ Imprimir / PDF

Usa el botón **"Imprimir / Guardar PDF"** o `Ctrl+P` desde el navegador. El diseño se adapta automáticamente para impresión en blanco y negro o color.

## 🔧 Tecnología

- HTML5 + CSS3 + JavaScript vanilla
- Sin dependencias externas
- Sin servidor ni base de datos
- Compatible con todos los navegadores modernos
- Funciona 100% offline

## 📊 Fuente del algoritmo

El algoritmo de cálculo fue extraído y verificado a partir de una hoja de cálculo Excel con tablas de correlación del calendario Maya Tzolkin, validado contra más de 40,000 fechas.
